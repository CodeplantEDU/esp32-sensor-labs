# 📟 LCD & OLED 디스플레이 + ESP32 실습 가이드

## 📌 개요

이 프로젝트는 **LCD(문자형 디스플레이)**와 **OLED(그래픽 디스플레이)**의
동작 원리를 이해하고,\
ESP32 + Thonny(MicroPython)를 활용하여 직접 제어해보는 실습입니다.

# 1️⃣ 통신 방식 개념

-   SDA, SCL 2선
-   주소 기반 통신

## 🔹 SPI

-   MOSI, MISO, SCK, CS
-   빠른 통신

# 6️⃣ 실습 코드

## LCD

``` python
from machine import Pin, I2C
from i2c_lcd import I2cLcd

i2c = I2C(0, scl=Pin(22), sda=Pin(21))
lcd = I2cLcd(i2c, 0x27, 2, 16)
lcd.putstr("Hello ESP32")
```

## OLED

``` python
from machine import Pin, I2C
import ssd1306

i2c = I2C(0, scl=Pin(22), sda=Pin(21))
oled = ssd1306.SSD1306_I2C(128, 64, i2c)
oled.text("Hello OLED", 0, 0)
oled.show()
```
